# Monetizasyon

Gelir modeli, fiyatlandırma, unit economics.

## Tipik dosyalar

- `gelir-modeli.md` — komisyon / abonelik / reklam / B2B karışımı ve önceliği
- `unit-economics.md` — CAC, LTV, marjlar, break-even, cohort analizi
- `fiyat-politikasi.md` — usta komisyon oranı, kullanıcı ücretleri, dikey bazında farklılaştırma (kaza / bakım / çekici / arıza)
- `reklam-modeli.md` — mevcut [../../feature-reklam.md](../../feature-reklam.md) BD uzantısı
- `b2b-fiyatlandirma.md` — filo + sigorta + kurumsal tarife

## İlke

Her gelir kalemi için: birim (kim ödüyor, ne karşılığında) + hacim tahmini (aylık vaka sayısı × ortalama ticket) + marj + riziko (disintermediation olasılığı). Agregat değil, dikey bazında.
