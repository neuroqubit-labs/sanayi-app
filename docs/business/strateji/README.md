# Strateji

Naro'nun uzun vadeli yönünü belirleyen dokümanlar.

## Tipik dosyalar

- `vizyon-tezi.md` — neden bu iş / neden şimdi / neden biz
- `kuzey-yildizi.md` — tek başarı metriği + destekçi KPI'lar
- `YYYY-Qn-plani.md` — çeyrek düzeyinde öncelik seti (örn. `2026-Q2-plani.md`)
- Tematik memolar: `anti-disinter-strateji.md`, `pasif-deger-tezi.md`, `dusuk-frekans-cozumu.md`

## İlke

Strateji dokümanı = tez + kanıt + sonuç. Tezin altında veriye/gözleme dayalı argüman, sonunda "bunu şu aksiyonla test ederiz / şu KPI ile ölçeriz" satırı.
