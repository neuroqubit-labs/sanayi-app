# Go-to-Market

Lansman planı, kanal stratejisi, segment/şehir yol haritası.

## Tipik dosyalar

- `lansman-plani.md` — v1 lansman senaryosu (pilot şehir, supply-first vs demand-first, hafta-hafta plan)
- `kanal-stratejisi.md` — SEO + sigorta + offline + referral + paid performans karışımı
- `soguk-baslangic.md` — iki taraflı marketplace chicken-and-egg çözümü (usta önce mi, müşteri mi?)
- `sehir-plani.md` — şehir bazlı açılım sıralaması + kriterler (araç yoğunluğu, sanayi yoğunluğu, dijital okuryazarlık)
- `segment-brief-<x>.md` — kadın kullanıcı, filolar, genç sürücü, yaşlı kullanıcı vb. segment odaklı GTM

## İlke

GTM bir lansman menüsü değil, **sınırlı kaynakla kazanma sırası**dır. Her kanalda CAC tahmini + ilk 90 gün hipotezi zorunlu.
