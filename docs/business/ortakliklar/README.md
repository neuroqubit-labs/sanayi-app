# Ortaklıklar

## Kontrol düzlemi

- **[pipeline.md](pipeline.md)** — canlı pipeline (kim, hangi aşamada, sonraki adım)

## Kategori memoları (eklenecek)

- `sigorta-ortakligi.md` — trafik/kasko entegre çekici + hasar dosyası
- `oem-yetkili-servis.md` — araç marka ortaklığı, garanti entegrasyonu
- `filo-b2b.md` — B2B ölçek, sözleşmeli kullanım (araç kiralama, kurumsal filo)
- `sanayi-birligi-kooperatif.md` — usta tarafı arz toplama (TESK, esnaf odaları)
- `odeme-baas.md` — escrow, sanal POS sağlayıcı
- `dagitim-kanali.md` — akaryakıt zincirleri, süpermarket, ruhsat yenileme noktaları
