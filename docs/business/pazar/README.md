# Pazar

Sektör, rekabet, kullanıcı tarafının derinlikli analizleri.

## Tipik dosyalar

- `pazar-tanimi.md` — TAM / SAM / SOM, pazar büyüklüğü tahmini, kaynaklar
- `sektor-dinamikleri.md` — talep kaynakları, fiyatlama, aracı zinciri, mevsimsellik
- `rakip-haritasi.md` — Yerel (MVG, Otosanal, Yedek Parça vs.) + uluslararası (RepairPal, YourMechanic) + yan rakip (sigorta app, yol yardımı) mukayesesi
- `kullanici-arastirmasi-YYYY-Qn.md` — interview + anket çıktıları, persona güncellemeleri
- `pazar-trendleri-YYYY.md` — regülasyon, EV penetrasyonu, filolanan araç oranı, sigorta kapsamı
- `ikincil-pazar-arastirmasi.md` — TÜİK, TSB, YHGS, OSD, TESK açık veri

## İlke

Her iddia kaynak referansı ile — rakam + tarih + link/yayın. Tahmin ise "tahmin" olarak etiketle.
